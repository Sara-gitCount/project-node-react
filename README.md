# 🛍️ Online Store

Full Stack e-commerce website developed with React and Node.js.

## 🚀 Technologies
- React
- Node.js
- Express
- MongoDB / SQL Server
- JWT Authentication

## ✨ Features
- User registration and login
- Display all products
- Shopping cart management
- Add and remove products from cart
- Product details page
- Responsive design
